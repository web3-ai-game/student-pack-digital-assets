## 7. 工具鏈配置 (JetBrains + GitHub + Sentry + Linear)

- **版本控制**：GitHub (每日 Commit, 開源 3+ Repo)。
- **錯誤監控**：Sentry (項目部署後即時追蹤 Bug)。
- **任務管理**：Linear (每日 Standup, 每月 Sprint)。
- **配置步驟**：
  1. 安裝 JetBrains Toolbox → 啟用 Student License。
  2. GitHub：創建 Organization，啟用 Codespaces。
  3. Sentry：免費帳號，整合 GitHub Actions。
  4. Linear：導入本計劃模板，每週自動提醒。
- **最佳實踐**：每日 Pull Request 習慣，Sentry Dashboard 作為項目健康指標。
