# GitHub Student Pack 學習路線圖與資產使用指南

## 1. 按技術棧分類的學習路線圖

### 🖥️ **Web 全棧開發路線**
**目標**：從前端到後端，掌握完整 Web 開發技能，適合初學者到進階開發者。

| 階段 | 推薦課程/資源 | 平台 | 預計時間 |
|------|--------------|------|----------|
| **基礎（1-2月）** | JavaScript、CSS、React 入門 | Frontend Masters (15門)、Educative (10路徑) | 4週 |
| **前端進階（2-3月）** | TypeScript、Node.js、進階 React | Frontend Masters (20門)、GitHub Skills (30連結) | 6週 |
| **後端全棧（3-4月）** | Node.js、Rails、Go/Rust | Frontend Masters (15門)、GoRails (全108集) | 6週 |
| **部署與實戰（4-6月）** | 完整專案、DevOps | Educative (15路徑)、GitHub Skills (50連結) | 8週 |

**推薦順序**：
1. Frontend Masters: JavaScript → CSS → React
2. Educative: Web 開發路徑
3. Frontend Masters: TypeScript + Node
4. GoRails: 全 Rails 教程
5. GitHub Skills: 實作專案

### 🤖 **AI-ML 路線**
**目標**：Python 數據基礎 → ML → AI 應用開發。

| 階段 | 推薦課程/資源 | 平台 | 預計時間 |
|------|--------------|------|----------|
| **數據基礎（1月）** | Python、SQL、R | DataCamp (30門) | 4週 |
| **ML 基礎（2-3月）** | 機器學習、深度學習 | DataCamp (40門)、Frontend Masters (AI課程) | 6週 |
| **AI 進階（4-5月）** | 生成式 AI、NLP | DataCamp (20門)、Educative (AI路徑) | 6週 |
| **專案實作（5-6月）** | AI 專案、部署 | JetBrains 教程 + GitHub Skills | 4週 |

**推薦順序**：
1. DataCamp: Python → SQL → 數據分析
2. DataCamp: ML 入門 → 深度學習
3. Frontend Masters: AI 相關課程
4. Educative: AI 學習路徑

### 🎮 **遊戲開發路線**
**目標**：Unity + Unreal Engine，快速上手遊戲開發。

| 階段 | 推薦課程/資源 | 平台 | 預計時間 |
|------|--------------|------|----------|
| **Unity 基礎（1-2月）** | Unity 入門教程 | Unity Learn (全12路徑) | 4週 |
| **Unity 進階（2-3月）** | 2D/3D 遊戲開發 | Unity Learn + Iconscout (3D資產) | 6週 |
| **Unreal 入門（3-4月）** | Unreal Engine 教程 | Unreal Engine (1教程) | 4週 |
| **完整遊戲（4-6月）** | 專案實作 + 資產整合 | 所有遊戲資源 | 8週 |

**推薦順序**：
1. Unity Learn: 基礎教程 → 中級路徑
2. Unreal Engine: 官方教程
3. 整合 Icons8/Iconscout 資產實作

### 🎨 **UI/UX 設計路線**
**目標**：設計基礎 → 原型製作 → 品牌設計。

| 階段 | 推薦課程/資源 | 平台 | 預計時間 |
|------|--------------|------|----------|
| **設計基礎（1月）** | 設計原則、顏色理論 | Canva 設計學校 | 3週 |

| **品牌設計（3-6月）** | 品牌套件、AI 工具 | Canva 品牌套件 + JetBrains UI教程 | 8週 |

## 2. 設計資產的最佳利用策略

### 📁 **資產分類與使用場景**
```markdown
┌─ Icons8 (5類)
│  ├─ 圖標 → App/Web UI 元素
│  ├─ 插圖 → 行銷/部落格
│  ├─ 照片 → 網站背景/展示
│  ├─ 音樂 → 影片/遊戲
│  └─ Lunacy → Figma 替代，團隊協作

┌─ Iconscout (47類)
│  ├─ Lottie 動畫 → 網站載入/互動
│  ├─ 3D 模型 → 遊戲/WebGL
│  └─ 插圖 → 產品展示

┌─ Canva (56類)
   ├─ 模板 → 快速製作簡報/社群
   ├─ 品牌套件 → 統一視覺識別
   └─ AI 工具 → 自動生成設計
```

### 🚀 **最佳實踐**
1. **專案啟動**：Canva 模板 → Icons8 圖標 → Iconscout 動畫
2. **遊戲開發**：Iconscout 3D + Icons8 音樂 + Unity 整合
3. **個人品牌**：Canva 品牌套件 + Icons8 照片建立作品集
4. **批量下載**：按專案需求分類儲存，避免混亂

## 3. 面試準備計劃（6個月）

### 📅 **分階段面試衝刺**
| 週數 | 重點 | 資源使用 | 每日時間 |
|------|------|----------|----------|
| **1-12週** | 技術基礎 | Frontend Masters + DataCamp | 2小時 |
| **13-18週** | 演算法練習 | Interview Cake (前75題) | 3小時 |
| **19-22週** | 進階題目 | Interview Cake (76-120題) | 4小時 |
| **23-24週** | 系統設計 | Educative 系統設計路徑 | 3小時 |
| **25-26週** | 模擬面試 | Interview Cake (剩餘題) + GitHub Skills | 4小時 |

**Interview Cake 使用策略**：
```
每日流程：
1. 早間：1-2道新題 (30-60分鐘)
2. 午間：複習錯題 (30分鐘)
3. 晚間：LeetCode 對應題目 (1小時)
4. 週末：完整模擬面試 (3小時)
```

### ⚡ **里程碑檢查點**
- **1個月**：完成基礎課程，建立 GitHub 作品集

- **5個月**：完成 100+ Interview Cake 題目
- **6個月**：準備 3-5 個作品集專案，開始投遞實習/工作

**💡 成功關鍵**：
1. 每天固定學習時間
2. 每週至少 1 個小專案 commit
3. 每月更新作品集
4. 加入 Discord/Reddit 學習社群
